Besplatan hosting za web stranicu - GitHub Pages

Većina hosting provajdera naplaćuje 20-50 KM mjesečno za hosting web stranice. Ali postoji alternativa koja je potpuno besplatna, brza i pouzdana. Zove se GitHub Pages i koriste je milioni sajtova širom svijeta - uključujući ovaj koji upravo čitate.

Šta je hosting i zašto ga trebate?

Hosting je prostor na internetu gdje "živi" vaša web stranica. Kada neko ukuca vašu adresu u preglednik, hosting server šalje fajlove vaše stranice tom posjetiocu. Bez hostinga, vaš sajt ne može postojati online.

Tradicionalno, hosting se plaća mjesečno - od 10 do 100+ KM, zavisno od provajdera i paketa. Ali za prezentacijske sajtove malih firmi, to je nepotreban trošak.

Šta je GitHub Pages?

GitHub Pages je besplatna usluga koju nudi GitHub - najveća platforma za razvoj softvera na svijetu, u vlasništvu Microsofta. Omogućava vam da objavite web stranicu direktno iz GitHub repozitorija.

Jednostavno rečeno: vi postavite fajlove svog sajta na GitHub, i oni ih besplatno servlraju posjetiocima. Bez plaćanja, bez reklama, bez ograničenja trajanja.

Šta dobijete besplatno?

  • Hosting - Potpuno besplatan, bez vremenskog ograničenja
  • SSL certifikat - HTTPS sigurnost uključena automatski
  • CDN - Vaš sajt se distribuira na servere širom svijeta za brže učitavanje
  • Custom domena - Možete povezati svoju domenu (npr. vasafirma.ba)
  • 99.9% uptime - Sajt je praktički uvijek dostupan
  • Automatski deploy - Svaka izmjena se automatski objavljuje

Da li je GitHub Pages dovoljno brz?

Da. GitHub Pages koristi globalni CDN (Content Delivery Network) koji distribuira vaš sajt na servere širom svijeta. Posjetilac iz BiH dobija sajt sa najbližeg servera u Evropi, posjetilac iz Amerike sa američkog servera.

Rezultat? Vaš sajt se učitava za manje od 1 sekunde. To je brže nego većina plaćenih hostinga. Ovaj sajt koji čitate se učitava za oko 0.8 sekundi - testirajte sami.

Da li je sigurno?

GitHub je u vlasništvu Microsofta - jedne od najvećih tehnoloških kompanija na svijetu. Koriste ga milioni developera i kompanija globalno, uključujući NASA-u, Spotify, i Netflix.

Vaš sajt na GitHub Pages ima:

  • Automatski HTTPS (SSL) - svi podaci su šifrirani
  • DDoS zaštitu - zaštita od napada
  • Automatske backupe - vaši fajlovi su sigurni
  • Nema baze podataka - nema šta da se hakuje

Statičan sajt na GitHub Pages je zapravo sigurniji od WordPress sajta na plaćenom hostingu, jer nema baze podataka, nema admin panela i nema plugin-ova koji se mogu kompromitovati.

Koji su limiti?

GitHub Pages ima ograničenja, ali za prezentacijske sajtove su više nego dovoljni:

  • 1 GB prostora - Dovoljan za stotine stranica sa slikama
  • 100 GB bandwidth mjesečno - Dovoljan za desetine hiljada posjeta
  • Samo statičan sadržaj - HTML, CSS, JavaScript. Nema PHP-a ili baza podataka
  • Nema e-commerce - Ako trebate web shop, trebat će vam plaćeni hosting

Za 95% malih biznisa koji trebaju prezentacijski sajt - ovo je više nego dovoljno.

Kad vam treba plaćeni hosting?

GitHub Pages nije za svakoga. Trebat će vam plaćeni hosting ako:

  • Trebate online shop (WooCommerce, Shopify)
  • Trebate korisnički login i bazu podataka
  • Trebate WordPress sa dinamičkim sadržajem
  • Trebate email hosting na vašoj domeni
  • Imate više od 100.000 posjeta mjesečno

Ako ništa od ovoga ne važi za vas - GitHub Pages je pravi izbor.
